# Basic front-end tools setup / Podstawowa konfiguracja narzędzi do front-endu

## Basic front-end tools setup.
### Requires:
- node.js - download from nodejs.org (LTS recommended)

### Contains:
- Grunt task runer - requires ground-cli global installation -> `npm install -g grunt-cli`,
task configuration is in the `Gruntfile.js` file
- BrowserSync - local server for synchronous testing of websites / web applications available with automatic refresh of views
- node-sass - a tool for compiling `.scss` files to` .css`
- PostCSS - a tool for transforming css files, mainly used for adding prefixes to the browser versions of the browser
- Uglify - a tool for obfuscating javascript code


## Recommended editors:
- Visual Studio Code,
- SublimeText 3

## Additional tools:
- ConEmu - overlay on the system console for Windows systems

## Podstawowe środowisko do prac front-endowych.
### Wymaga:
- node.js do pobrania z nodejs.org (zalecana wersja LTS)

### Zawiera: 
- Grunt task runer - do działania wymaga globalnej instalacji grunt-cli -> `npm install -g grunt-cli`,
konfiguracja zadań jest w pliku `Gruntfile.js`
- BrowserSync - lokalny server do synchronicznego testowania stron www/aplikacji webowych dostępny z automatycznym odświeżaniem widoków
- node-sass - narzędzie do kompilacji plików `.scss` do `.css`
- PostCSS - narzędzie do transformacji plików css, głównie używany do dopisywania prefixów dla strszych wersji przeglądarek
- Uglify - narzędzie do zaciemniania kodu javascript


## Polecane edytory:
- Visual Studio Code,
- SublimeText 3

## Dodatkowe narzędzia:
- ConEmu - nakładka na systemową konsolę dla systemów Windows
